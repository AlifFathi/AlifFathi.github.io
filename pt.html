Programming Technique I

This course equips the students with theory and practice on problem solving techniques by using the structured approach. Students are required to develop programs using C++ programming language, in order to solve simple to moderate problems. The course covers pre-processor directives, constants and variables, data types, input and output statements, control structures: sequential, selection and loop, built-in and user-defined functions, single and two dimensional arrays, file operations, pointers, and structured data types.


Reflection:

I discover that Programming Technique I is about the method, reasoning, and technique a programmer uses to independently and swiftly create a software. A programmer may write code that performs a given set of calculations quickly, can draw logical conclusions, and can repeat a set of instructions exactly as written.
